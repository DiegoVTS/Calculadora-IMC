# Calculadora IMC

Calculadora de Índice de Massa Corporal desenvolvida com HTML, CSS e JavaScript puro.

## 📋 Sobre o Projeto

Projeto desenvolvido para praticar manipulação do DOM, lógica condicional e SVG animado com JavaScript. O usuário insere peso e altura e recebe o IMC, classificação e um velocímetro visual com ponteiro animado.

## ✨ Funcionalidades

- Cálculo de IMC em tempo real
- Classificação automática — Abaixo do peso, Normal, Sobrepeso, Obesidade
- Velocímetro SVG com ponteiro animado
- Mensagem de erro para campos inválidos
- Layout responsivo

## 🛠️ Tecnologias

- HTML5 Semântico
- CSS3 — Flexbox, Grid, responsividade
- JavaScript — DOM, eventos, SVG, lógica condicional
- Google Fonts — Poppins e Inter
